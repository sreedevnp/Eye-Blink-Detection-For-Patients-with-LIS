# Eye Blink Detection System for Patients with Locked-in Syndrome

A real-time computer vision based system developed using Python, OpenCV and Dlib to detect eye blinks and assist patients with communication and emergency alerts.

## Features
- Real-time eye blink detection using facial landmarks
- Emergency alert trigger when eyes remain open for long duration
- Prebuilt message communication system
- Food menu and emergency service interface
- Audio alert support

## Technologies Used
- Python
- OpenCV
- Dlib
- Computer Vision
- Facial Landmark Detection

## Setup Instructions
1. Clone the repository
2. Install required libraries:
   pip install opencv-python dlib numpy
3. Download shape_predictor_68_face_landmarks.dat from Dlib official site
4. Place it in the project root folder
5. Run:
   python eyeblink.py

## Use Case
Designed to help patients with Locked-in Syndrome communicate and trigger emergency alerts using eye movement.
